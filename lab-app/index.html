Claude Code

A Hands-On Primer

A hands-on session to understand Claude Code — install it, explore its ecosystem, and master the primitives that turn it from a chat app into an agent runtime you actually command.

RSVP on Meetup View on GitHub View materials

What you'll learn

  • Install & first run

    Get Claude Code running, authenticated, and responsive to your codebase.

  • Memory hierarchy

    CLAUDE.md layers, path-scoped rules, auto-memory — what Claude remembers and when.

  • Running modes & permissions

    Plan mode, headless, the four permission modes, and the settings.json allow/deny/ask model.

  • Worktrees + parallel Claude

    Run multiple Claude sessions on one repo without collisions. Fork conversations safely.

  • The four extension primitives

    Slash commands, skills, subagents, hooks — when to reach for which, and how to build each.

  • Live walkthrough of a real skill

    A 285-line /tutor skill plus its companion research subagent. Yours to take home.

Who should come

  • Engineers who've tried Claude Code and want depth past the primer
  • AI-curious developers looking for practical patterns, not marketing
  • Team leads considering rolling Claude Code out to their teams

Prerequisites

Git, a terminal, and a Claude account (or an ANTHROPIC_API_KEY). Install instructions live at code.claude.com/docs/en/quickstart.

What you'll leave with

  • Claude Code installed and configured for your workflow
  • A working mental model of its ecosystem
  • The /tutor skill + tutor-researcher subagent dropped into your ~/.claude
  • A 27-chapter master-class companion guide for deeper self-study
